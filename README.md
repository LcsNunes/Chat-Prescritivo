# Chat Prescritivo

Projeto simples de manutencao prescritiva com IA Generativa.

O objetivo e analisar eventos do `banner.csv`, localizar eventos historicos similares, recuperar documentos tecnicos por RAG e gerar uma resposta prescritiva com uma LLM local via Ollama.

Mais detalhes serao adicionados conforme as etapas do projeto forem implementadas.
